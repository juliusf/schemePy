Holodeck Scheme REPL

Simple Scheme interpreter written in python

Introduction & Usage

For a lecture, I've implemented an interpreter for a subset of the Scheme programming language in python. The sources for the interpreter as well as for the web application server are available on my github profile. Usage is quite easy: Scheme expressions entered in the text editor labled "File" can be evaluated by clicking on the "Evaluate File" button. Saved variables and functions are lost every time the button is clicked. After a file has been evaluated the values in the enviornment can still be accessed via the Direct Window. Expressions in this window can be evaluated by clicking "Evaluate Direct Window", respectively.

Supported Features

Supported Scheme Syntax

  • define
  • begin
  • lambda
  • if
  • cons
  • car
  • cdr
  • let
  • set!
  • quote
For convenience lambda shorthand syntax as well as implicit begin statements and comments (beginning with ";") are supported.

Builtin Function

  • +
  • -
  • *
  • /
  • =
  • >
  • >=
  • <
  • <=
  • and
  • or
  • not
